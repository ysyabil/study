## 2 文本标签

| 标签名 | 语义和功能                           | 属性 | 单标签还是双标签 |
| ------ | ------------------------------------ | ---- | ---------------- |
| em     | 表示强调，通常表现为斜体字           |      | 双               |
| strong | 表示强调（更强烈），通常表现为粗体字 |      | 双               |
| del    | 表示被删除的内容，表现为文字加删除线 |      | 双               |
| ins    | 表示新添加的内容，表现为文字加下划线 |      | 双               |
| sub    | 下标字                               |      | 双               |
| sup    | 上标字                               |      | 双               |
